What Skills Should You Learn?

And what skills can I teach you?

Programming

     One of the twenty-first century skills I like to encourage people to learn is programming. There are many programming languages out there and I myself am currently producing an ongoing Python Course which is an intuitive beginners programming language. I think people should learn how to program because it engages a part of our brain that we don't utilize much in our society, creative problem-solving, with code you can use computers to their fullest potential, and bend there capabilities to your will. Theres nothing you can't do.

3D Art

     The animation industry is heading towards full 3D Animation, less and less classical animation is used. This isn't a neccesarily a bad thing, but it does mean that traditional artists, and animators need to learn to adapt to out technological world. Additionally, despite indie games having a reputation for soley using 2D pixel art, more and more indie game developers are able to compete with triple games, in the 3D environment.

Game Development

     No, I don't have a valid argument for why you should learn game development for practical purposes. However I can say with absolute certainty that it is a fun way to hone your programming and creative problem-solving skills.
